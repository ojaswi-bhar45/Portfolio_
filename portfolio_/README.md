# 🚀 Ojaswi Bhardwaj - Portfolio

A modern, responsive portfolio website showcasing my skills and projects as a Full Stack Developer. Built with React, Vite, and Tailwind CSS, featuring smooth animations powered by Framer Motion and a stunning purple/pink gradient design.

## ✨ Features

- **Modern Design**: Sleek, contemporary layout with purple/pink gradient color scheme
- **Centered Hero**: Profile image prominently centered with elegant typography
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: Beautiful animations using Framer Motion with 3D effects
- **Dark Theme**: Modern dark background with vibrant accent colors
- **Fast Performance**: Built with Vite for lightning-fast development and builds
- **SEO Optimized**: Proper meta tags and semantic HTML

## 🛠️ Tech Stack

### Frontend
- **React 18** - Modern React with hooks and functional components
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library for React

### Development Tools
- **ESLint** - Code linting
- **PostCSS** - CSS processing
- **Autoprefixer** - CSS vendor prefixing

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ojaswi-bhar45/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   
   Navigate to `http://localhost:5173` to view the portfolio.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 🎨 Customization

### Color Theme

The portfolio uses a modern purple/pink gradient theme. You can modify the colors in `tailwind.config.js`:

```javascript
colors: {
  'primary': '#6366f1',      // Indigo
  'secondary': '#8b5cf6',    // Purple
  'accent': '#ec4899',       // Pink
  'dark-bg': '#0f0f23',      // Dark background
  'card-bg': '#1a1a2e',      // Card background
  'text-primary': '#ffffff', // White text
  'text-secondary': '#cbd5e1', // Light gray text
}
```

### Content Updates

Update your personal information in `src/App.jsx`:

- **Personal Links**: Modify the `actionLinks` array
- **Skills**: Update the `skills` array
- **Projects**: Edit the `projects` array with your own projects
- **Profile Image**: Replace `src/assets/profile.jpg` with your photo
